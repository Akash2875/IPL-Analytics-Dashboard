# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

## 📖 Overview

The **IPL Analysis Dashboard (2008–2025)** is a dynamic and interactive Power BI dashboard designed to analyze the Indian Premier League across multiple seasons. The dashboard provides comprehensive insights into team performances, player statistics, tournament outcomes, and season-wise trends through interactive visualizations and dynamic filtering.

This project transforms raw IPL data into meaningful insights, enabling users to explore historical records, compare team and player performances, and understand tournament patterns from **2008 to 2025**.

---

## 🎯 Problem Statement

The Indian Premier League (IPL) has generated a massive volume of data since its inception in 2008, including match results, team performances, player statistics, venue information, and tournament records. While this data contains valuable insights, analyzing it across multiple seasons can be challenging due to its complexity and scale.

Cricket analysts, fans, and stakeholders often face difficulties in:

- Tracking team performances across different seasons.
- Identifying top-performing players.
- Comparing batting and bowling achievements.
- Understanding tournament trends and statistics.
- Analyzing season-specific outcomes efficiently.
- Extracting meaningful insights from large datasets.

To address these challenges, this dynamic Power BI dashboard was developed to consolidate IPL data from **2008–2025** into a single interactive platform. The dashboard enables users to analyze team standings, player achievements, season statistics, and tournament outcomes through intuitive visualizations and dynamic filtering.

- Wins
- Losses

- Highest Run Scorer

- Purple Cap Holder

- Teams

Allows users to select any season from **2008–2025**, automatically updating all visuals and KPIs.
